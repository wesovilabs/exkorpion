Installation

If you are here is because you are an Elixir developer who wants to learn about Exkorpion framework. So, we assume you already have a ready environment to code in Elixir, If not we encourage you to have a look at the links on the left side

Library is available in Hex, the package can be installed as:

  • 1. Add exkorpion to your list of dependencies in mix.exs: 
     def deps do 
       [{:exkorpion, "~> 0.0.2"}]
      end
  • 2. Ensure exkorpion is started before your application: 
     def application do 
       [applications:[:exkorpion]]
     end
