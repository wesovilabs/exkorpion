Overview

Exkorpion wraps ExUnit and enhance their features providing developers with a BDD syntax

One of the goals of Exkorpion is that makes tests easy to be read..

What make Exkorpion different

Often developers claim about the benefits of the functional programming. Would you any time imagine on bring these good things to the test side? Exkorpion force us to write functions for all the steps in our tests. Making use of anonymous function Exkorpion achieves a way to make developers think (even more) about functions in a functional programming language.
